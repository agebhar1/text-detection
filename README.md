# text-detection
very simple text detection using opencv and python <br>
也可以用于中文！ <br>
심지어 한국 <br>

# usage
``` bash
python TextDetect.py input.png
```

# requirements
* Python > 3.0 (Tested with: 3.4.3)
* Opencv > 3.0 (Tested with: 3.0.0)
